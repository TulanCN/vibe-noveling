# 剧情设计工具箱

8 个工具为参考定义。场景设计阶段根据场景内容自动匹配并应用，不需要用户确认。

## 自动应用规则

场景设计完成后，根据以下条件自动判断适用的工具，将工具技巧直接融入场景描述和写作提示中：

| 工具 | 自动触发条件 |
|------|-------------|
| A 情绪递进五步法 | 场景包含打脸/逆袭/被轻视的情节 |
| B 锁式危机设计 | 场景有紧迫时间/空间/资源限制 |
| C 大小反差设计 | 角色身份与所处环境有反差 |
| D 信息差分层释放 | 涉及金手指/身份/秘密揭示 |
| E 道德挑战设计 | 角色面临两难选择 |
| F 围观震惊 | 主角公开展现实力/能力 |
| G 实力碾压 | 有战斗/实力对比场景 |
| H 装逼打脸四步节奏 | 与 A 同时触发（五步铺垫 → 四步爆发）|

多个工具可以叠加使用。

## 工具 A：情绪递进五步法（打脸/逆袭类必备）

打脸场景不是一步到位的，必须层层递进拉满读者情绪：

```
五步递进法：
   第一步：上门被拒 → 轻视/无视主角
   第二步：冷眼相待 → 当众嘲讽/刁难
   第三步：当众羞辱 → 触及主角底线/尊严
   第四步：引入对比 → 对同样的事对富二代/关系户热情
   第五步：强迫选择 → 逼迫主角做出让步/牺牲

✅ 五步到位后打脸的爽感，是一步到位的五倍。

⚠️ 注意：不是每个打脸都需要全部五步。根据场景时间选择 3-5 步，
     但至少 3 步才能拉够情绪。
```

## 工具 B："锁"式危机设计

通过设置"锁"来瞬间提升紧迫感：

- **时间锁**：主角必须在限定时间内完成某事（"三炷香内不破阵，阵法自毁"）
- **空间锁**：角色被困于有限空间（暴风雪山庄、密室、孤岛）
- **资源锁**：关键资源有限（只有一枚破阵符、灵力只够一次攻击）

**核心效果**：锁迫使角色做选择，选择暴露性格，推动剧情。

## 工具 C："大小反差"设计

利用身份与环境的极致反差制造天然戏剧性：

- **大人物遇小事**：强者处理日常琐事 → 降维打击的爽感 / 水土不服的笑点
- **小人物遇大事**：平凡主角卷入重大事件 → 读者为他如何脱困而揪心

```
✅ 大人物遇小事：
   凌绝尘站在菜市场里，面前摆着三个灵菇。
   摊主："五个灵石一个。"
   凌绝尘："……太贵了。"
   摊主："这还贵？你去别家看看。"
   凌绝尘没有还价的经验。
   他沉默了三秒。
   "……三个。"

✅ 小人物遇大事：
   苏寒染只是在通讯塔上修设备。
   他不知道塔下面，三个宗门长老正在讨论他的处置方案。
```

## 工具 D：信息差分层释放

主角的秘密（金手指/身份/实力）不一次揭穿，而是分层释放，每一层都是一个独立爽点：

```
信息差分层模板：
   第一层：室友/队友 → 偶然发现异常
   第二层：同组/熟人 → 进一步震惊
   第三层：权威/长老 → 重新评估主角
   第四层：对手/反派 → 恐惧/重视
   第五层：更高层面（宗主/大能） → 布局级别的关注

⚠️ 关键：在旧信息差即将揭晓前，铺垫新的、更高级的信息差，
   让读者期待感无缝转移到新目标上。

示例：
   队友震惊于苏寒染会修阵法 → 同时暗示他的阵法可能来自失传的古法
   → 读者想知道：古法从哪来的？
```

## 工具 E："道德挑战"设计

设置让角色在道德、利益、情感之间做两难选择，极致展现人物内心并调动读者情绪：

```
道德挑战模板：
   核心结构：两个选项都有代价，无论怎么选都会失去什么

   设计要点：
   1. 选项 A 和选项 B 各有合理的理由（不是"善良 vs 邪恶"）
   2. 两个选项的代价要让读者也纠结
   3. 角色的选择必须符合其行为底层逻辑（不能为了剧情强行选）
   4. 选择的结果要有后续影响（不是选完就翻篇）

✅ 好的道德挑战：
   "救队友但暴露金手指，还是袖手旁观保全秘密？"
   → 两个选项都有道理，读者也不确定该怎么选

   "用不义之法获取拯救至亲的力量，还是坚持底线接受失去？"
   → 情感和原则的冲突，没有正确答案

❌ 差的道德挑战：
   "杀无辜的人还是不杀？"
   → 太明显的善恶对立，没有纠结空间
```

## 工具 F：围观震惊（群体反应计量）

配角不是道具，是主角牛逼的"计量器"。设计爽点时，不仅要设计事件本身，还要设计**不同层级角色的反应链**。

```
围观震惊层级设计：

第一层：执行者/围观平民（当场反应）
   → 惊呼、倒吸凉气、道具掉落
   → 设计要点：反应要具体，不能只写"震惊"

第二层：亲友/同僚（情感反应）
   → 沉默、握紧拳头、眼神变化、不愿相信
   → 设计要点：用沉默代替惊呼，更有力量

第三层：权威/高层（战略反应）
   → 放下手中的事、互相交换眼神、做出决定
   → 设计要点：用动作暗示态度转变

✅ 核心：一次铺垫收获 3-5 次爽点爆发，事半功倍。
   沉默比惊呼更有力量。动作比语言更有冲击。
```

## 工具 G：实力碾压（数值化/视觉化设计）

设计实力展示场景时，不要只写"很强"。设计三个维度让读者**看到**强到什么程度：

```
实力碾压设计三维度：

维度 1：对比法
   → 与已知参照物对比（修炼年限、资源消耗、境界差距）
   → 设计要点：用具体数字制造冲击（三十年 vs 三年）

维度 2：画面法
   → 用物理效果展示力量（空气撕裂、石壁碎裂、地面龟裂）
   → 设计要点：写具体的位置和范围（五丈外、拳头大小）

维度 3：反应法
   → 通过权威人物的微动作展示实力等级
   → 设计要点：用"放下手中的事"暗示高度重视
```

## 工具 H：装逼打脸四步节奏

```
标准节奏：压抑 → 升级 → 爆发 → 余韵

第一步：压制（制造不爽）
   → 对方看不起主角、嘲讽、阻挠
   → 周围人附和，让读者也憋着一口气
   → 主角表面不动声色，但读者知道他很强

第二步：升级（继续加压）
   → 对方得寸进尺
   → 主角的底线被触碰
   → 读者开始期待反击

第三步：爆发（一口气释放）
   → 主角出手，干净利落
   → 对方震惊/恐惧
   → 周围人倒吸一口凉气

第四步：余韵（让爽感持续）
   → 事后反应：众人议论、重新评估
   → 对方的挫败感
   → 主角云淡风轻，不把刚才的事当回事

⚠️ 注意：四步节奏与工具 A（情绪递进五步法）互补。
   五步法是铺垫层（被拒→冷眼→羞辱→对比→强迫选择），
   四步节奏是爆发层（压制→升级→爆发→余韵）。
   高潮章节可以叠加使用：五步铺垫 → 四步爆发。
```
